**UndertaleModTool** (**UMT** or **UTMT**) is a tool for modifying GameMaker IFF files created by krzys_h and maintained by contributors.

UndertaleModTool providers a number of features centered around modifying the data stored within GameMaker IFF files, including a compiler and decompiler for GML as well as interactive editors for GML, the underlying bytecode, and various other components such as assets.
